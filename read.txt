[Setting] - путь = appdata\lib\plugin.ini

Setup - Название проги внутри сетапа
Software - Название файла внутри папок (с расширкой)
Save = Название файла, когда он перемещается на рабочий стол (с расширкой)

Er - 0 = Установка завершится без ошибок | 1 = В конце установки будет писать об ошибке
TxEr - Текст ошибки
Launch - 0 = Не запускает/открывает файл после завершения "установки" | 1 = Принудительно (не смотря на то, стоит ли галочка вконце) запускает/открывает файл после завершения "установки"
